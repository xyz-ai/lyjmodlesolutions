# Design Skill - Industrial B2B Website

## Design Style
- Clean
- Minimal
- Industrial
- Export business focused
- Professional, not decorative

## Layout Rules
- Large hero section
- Clear section separation
- Generous whitespace (60–100px vertical spacing)
- Max content width around 1200px
- Avoid crowded layouts

## Typography
- Simple font hierarchy (heading + body)
- No excessive font sizes
- Comfortable line height

## Colors
- White background
- Dark blue / gray text
- One accent color (blue or orange)
- No gradients
- No random color mixing

## Components
- Card-based sections
- Clean buttons
- Consistent icon style
- Grid layout for products

## Content Rules
- Clear business structure
- No template filler text
- No fake testimonials or fake blog content
- Focus on real business value

## Avoid
- Old Bootstrap template look
- Car service template style
- Over-animation
- Cluttered icon blocks
- Fake corporate sections

## Goal
Make the website look like a real manufacturing/export company, not a demo template.
